# Apex Discord Bot

An advanced all-in-One Discord bot with 64+ commands.

## Quick Start (One Command)

```bash
cd /home/runner/workspace/discord-bot && npm start
```

## Features

- **Moderation** (16): kick, ban, unban, softban, mute, unmute, tempmute, warn, warnings, clear, slowmode, lock, unlock, nick
- **Information** (10): help, ping, serverinfo, userinfo, avatar, roleinfo, roles, invite, emojis, channelinfo, uptime
- **Utility** (16): search, scrape, afk, remind, poll, math, qr, shorten, encode, decode, binary, ticket, ticketclose, tickets, purge
- **Fun** (15): 8ball, dice, choose, cat, dog, ascii, ship, joke, meme, quote, fact, rps, flip, slots, rate, hack, love
- **Level System**: XP, levels, leaderboard
- **Admin** (3): say, announce, embed
- **Giveaways**: start, reroll

## Configuration

Edit `.env` with your tokens:
- `DISCORD_BOT_TOKEN` - Your Discord bot token
- `FIRECRAWL_API_KEY` - For web search/scraping (optional)

## Run

```bash
npm start
```

## Commands

Use `!help` to see all commands.
